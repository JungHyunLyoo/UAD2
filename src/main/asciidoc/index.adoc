[[overview-domain]]
=== 도메인

|===
| 환경 | 도메인

| 운영(version 1)
| `https://유아들.kr`
|===

[[overview-http-verbs]]
=== HTTP 메소드

|===
| 메소드 | 설명

| `GET`
| 리소스 검색

| `POST`
| 리소스 생성

| `PATCH`
| 리소스 부분 수정

| `DELETE`
| 리소스 삭제
|===

[[overview-http-status-codes]]
=== HTTP 응답 코드

|===
| 응답 코드 | 설명

| `200 OK`
| 요청 성공적으로 수행.

| `201 Created`
| 리소스 생성 완료.

| `204 No Content`
| 기존 리소스 수정 완료.

| `400 Bad Request`
| 잘못된 요청.

| `404 Not Found`
| 요청한 리소스 존재하지 않음.
|===
